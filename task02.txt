Программа аннотирования списка функций.
1. Программа должна построчно определять комментарии в стиле doxygen и при нахождении тега \func выводить описание функции в стандартный вывод.

/**
	\func <описание функции>
*/

/** \func int main() */

2. Входные данные 
Текст программы в файле

3. Выходные данные
Список функций вида:
1. <описание функции 1>
2. <описание функции 2>
...
N. <описание функции N>

4. Пример
Входные данные:
#include <stdio.h>

/** \func int sum(int a, int b) */
int sum(int a, int b) {
	return a+b;
}

/**
	\func int main()
*/
int main()
{
	return 0;
}

Выходные данные:
1. int sum(int a, int b)
2. int main()
